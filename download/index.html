STAR CITIZEN TRACKER

COMBAT ANALYTICS FOR STAR CITIZEN

HOW IT WORKS

Monitors Star Citizen's Game.log file for changes, providing real-time updates on player actions and events as an overlay. And Optionally uploads data to the database for stat tracking.

  • Highly Customizable
  • Tracks a variety of console outputs
  • Positionable & customizable overlay
  • Statistics tracking
  • No-Overlay mode for database tracking only
DETAILED STATISTICS

This translates a wide variety of console outputs into easily glanceable statistics.

  • K/D Tracking
  • Nearby Player deaths
  • Nearby ship attacks
  • Tracks weapon and damage types
  • Killstreak tracking
  • Player performance metrics
STREAMER READY

Built for content creators with OBS Browser integration and customizable overlays. You can alternatively also use your own tracker and leave this as background tracking for the database.

  • OBS browser source support
  • No-Overlay mode for database tracking only
  • Customizable Tracking Metrics
  • Hosted locally
